* для того, чтобы сделать коммит необходио выполнить две команды:

1) $ git add -A (добавляет все изменённые файлы в индекс)
2) $ git commit -m "Commit message." (создаёт коммит)
    ! если нужно добавить определённые, необходимо указать перед командой добавляемые в индекс файлы через пробел 

* создание коммита одной командой (только для отслеживаемых файлов (tracked files)): 
    $ git commit -a -m "Commit message."

* отправка изменениё на удалённый репозиторий:
    $ git push
    $ git push <имя отдалённого репозитория> master (отправляет все изменения в главной ветке (master))
